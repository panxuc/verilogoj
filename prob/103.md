# 七段译码器

## 题目描述

七段显示译码器，可以把BCD码转换为七段显示码，从而可以显示数字0至数字9。

对应关系为：

0->0111111

1->0000110

2->1011011

3->1001111

4->1100110

5->1101101

6->1111101

7->0000111

8->1111111

9->1101111

其他输入->0000000

要求：在always块中使用if-else或case语句实现。

## 输入格式

输入为 `din`，为 4-bit wire。

## 输出格式

输出为 `dout` 从BCD转为七段显示码的结果，位宽为 7。

## 代码编辑

```verilog
module BCD7( 
    input [3:0] din, 
    output [6:0] dout
);
    // 在这里输入你的代码 请不要修改模块和信号名称
endmodule
```
